# vikash-ai-portfolio

Personal developer portfolio with AI-powered content features. Built with React, TypeScript, and the Claude / OpenAI API as part of an internship project.

---

## Tech stack

- **Frontend** — React 18 · TypeScript · Vite
- **Styling** — Tailwind CSS · shadcn/ui
- **AI** — Anthropic Claude API / OpenAI API

---

## Getting started

```bash
git clone https://github.com/vikash1311/vikash-ai-portfolio.git
cd vikash-ai-portfolio
npm install
npm run dev
```

Add your API key to a `.env` file:

```
VITE_ANTHROPIC_API_KEY=your_key_here
```
